# 🌙 Luna TV Website

ئەم وێبسایتە بۆ **لونا تیڤی (Luna TV)** دروست کراوە  
بۆ نیشاندانی ناوەڕۆک و بەڕێوەبردنی خزمەتگوزارییەکان بە شێوەیەکی سادە و خێرا.

---

## 🚀 تایبەتمەندییەکان
- ✅ وێبسایتی مۆدێرن و خێرا  
- ✅ دامەزراندن بە Node.js  
- ✅ ئامادە بۆ دیپلۆی کردن لە VPS / Hosting  
- ✅ گونجاو بۆ بەکارهێنەران  

---

## 🛠️ چۆنیەتی دیپلۆی کردن

### 1️⃣ دامەزراندنی پەکیجەکان
```bash
npm i
```

### 2️⃣ دروستکردنی فایلەکان (Build)
```bash
npm run build
```

### 3️⃣ کارپێکردنی وێبسایت
```bash
npm run start
```

---

## 📦 فەرمانە سەرەکییەکان
- `npm i` → دامەزراندنی گشت پێداویستییەکان  
- `npm run build` → دروستکردنی فایلە ئامادەکان  
- `npm run start` → کارپێکردنی وێبسایت
